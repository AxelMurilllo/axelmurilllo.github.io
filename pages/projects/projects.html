Marginalia Magic

Project Overview

A modern React-based editor for Melville's Marginalia Online (MMO), replacing their outdated Adobe Flash/Flex markup editor. This tool helps catalog Herman Melville's personal library by converting his handwritten notes and markings into a machine-readable digital format.

Key Features

  • Document loading and management from MMO database
  • Page-by-page navigation and annotation
  • Interactive bounding box creation for text/image annotations
  • XML export functionality for offline use
  • Modern, intuitive user interface

Impact

This project ensures MMO remains a sustainable and accessible resource for scholars, researchers, and literary enthusiasts interested in understanding Melville's annotations and creative process. The modernized interface and improved functionality contribute to the long-term preservation of Melville's literary annotations in a sustainable digital format.
